摘要：关于在国内如何快速打开github


第一步：查询github.com目前的ip地址

打开ipaddress.com，然后在搜索栏搜索github.com，找到红色框圈住的就是目前的ip地址。

可以看到，在我写这篇博客的时候，github.com的ip为:140.82.114.3
